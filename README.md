# Python3-AmazonRankingScraper
## Prerequisite

python >= 3.8

## Installation

### linux

```
$ python -m venv venv
$ . ./venv/bin/activate
$ python -m pip install --upgrade pip
$ pip install -r requirements.txt
```

### Mac

```
$ python3 -m venv venv
$ . ./venv/bin/activate
$ python3 -m pip3 install --upgrade pip
$ pip3 install -r requirements.txt
$ 
```

## How to use

```
python3 amazon_ranking.py    
```
